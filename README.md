**Bus.lk** is a full-stack smart bus platform developed as a university project to improve the passenger experience in public transportation. The system focuses on solving real-world commuter problems using real-time technologies, scalable backend architecture, and a user-friendly interface.



## Problem Statement
Public bus passengers commonly face daily challenges:

- Uncertainty about bus arrival times  
- Long waiting periods at bus stops  
- Difficulty identifying correct routes and bus numbers  
- No visibility of seat availability  
- Manual ticket purchasing processes  
- Limited or no digital payment methods  
- Lack of centralized travel information  

These issues reduce efficiency, waste time, and create frustration for commuters.



##  Our Solution
**Bus.lk** provides a centralized digital platform designed specifically for passengers. It introduces modern smart features to make bus travel easier, faster, and more reliable.

In a real-world implementation, buses would be equipped with IoT GPS devices that continuously send live location data to the server. Passengers could then track buses in real time and arrive exactly when the bus reaches their stop, reducing unnecessary waiting time.

For this prototype, we implemented a **simulated tracking system** to demonstrate how real-time tracking would function in production.
